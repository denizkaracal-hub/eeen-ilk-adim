# EEEN Mühendislik Yol Haritası
2026 yılı itibariyle teknik mülakatlar ve kendimi geliştirmek için bu depoda projelerimi toplayacağım.
LTspice ile 1.6kHz kesim frekanslı alçak geçirgen filtre simüle ettim
## Python ile Veri Analizi
* `filtre_analiz.py` scripti ile tasarladığım filtrenin farklı frekanslardaki V_out ve dB değerlerini matematiksel olarak hesaplayan bir otomasyon aracı yazdım.
## Dijital Sinyal İşleme (DSP) Uygulaması
* Osiloskoptan alınan 5 kHz gürültülü sinyali, Python `scipy` kütüphanesi kullanarak tasarladığım 1.6 kHz kesim frekanslı dijital Butterworth filtresinden geçirdim ve gürültüyü tamamen temizledim.
